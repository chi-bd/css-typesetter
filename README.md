# CSS-Typesetter

GUI Tool for typesetting with CSS.

## Usage

[tohashi.github.io/css-typesetter](http://tohashi.github.io/css-typesetter/)

or

```sh
$ git clone git@github.com:tohashi/css-typesetter.git
$ cd css-typesetter
$ npm install
$ npm run watch
```

See [localhost:3333](http://localhost:3333/).
